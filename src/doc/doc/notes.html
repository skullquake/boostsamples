Boost provides a set of useful libraries that makes programming c++ much easier. Some of the boost libraries are headers only and some are compiled libraries, either static or shared, depending on how they were built. The built libraries include:

  • Boost.Chrono: This is used for clock operations and calculating durations
  • Boost.Context: This is used to create higher-level abstractions like coroutines and cooperative threads.
  • Boost.Filesystem: This is used to deal with files and directories.
  • Boost.GraphParallel: This extends Boost Graph Library (BGL) and is used for parallel and distributed computing.
  • Boost.IOStreams: This is used to write and read data using streams. You can for example use this to load a file into memory or write compressed data in GZIP format.
  • Boost.Locale: To localize applications into other user languages.
  • Boost.MPI: This is used to create programs that executes tasks concurrently. It stands for Message Passing Interface.
  • Boost.ProgramOptions: This is used to parse command-line options.
  • Boost.Python: Used to parse the Python language in C++ code.
  • Boost.Regex: This is used to apply regular expressions.
  • Boost.Serialization: To convert objects into series of bytes for saving to disk or sending over a network for example.
  • Boost.Signals: To create signals, to which functions will respond and execute.
  • Boost.System: This is used to define errors. It contains four classes: boost::system::error_code, boost::system::error_category, boost::system::error_condition and boost::system::system_error.
  • Boost.Thread: This is used for threaded programming. Classes for synchronized access on multiple-thread data are provided.
  • Boost.Timer: This is used to measure code performance using clocks.
  • Boost.Wave: Provides a reuasable C preprocessor that can be used in C++ code.

There are also some libraries that have optional, seperately compiled binaries:

  • Boost.DateTime: Used to process time and data data.
  • Boost.Graph: Used for 2D graphics.
  • Boost.Math: Deals with math formulas.
  • Boost.Random: This is used to generate random numbers.
  • Boost.Test: This is used to write and organize test programs.
  • Boost.Exception: Used to add data to an exception after it has been thrown.

Boost provides lots of useful features not available in STL, as well as some alternatives to work around some of STL's implementations or designs. It also has excellent documentation with examples.

Most of the core classes are templates, and to use them you need to include the appropriate set of header files. Also, the compiled libraries are rather large, and actually not feasible for smaller projects, and can add to compile times.
