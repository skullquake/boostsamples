Here we look at representing date objects in c++ using Boost

Boost provides a type to represent date in gregorian calendar
There are different ways of initializing such a date object
boost::gregorian::date(1990,1,1);
boost::gregorian::date("1990-01-01");
boost::gregorian::from_undelimited_string("19900101");

You can access the year, month, or day attributes using the year(), month(), and day() member functions of a date object. The day of week can be accessed using the day_of_week() member function

There is also a way to initialize a date object with todays date instead of a hardcoded date or a date generated or date read from a database (try this out with Postgres or MySQL)

You can initialize a date object with current date in UTC Timezone using the universal_day() function
